A tiny switch with a whole sky inside.

Skyshift brings an illustrated day-and-night theme toggle to any website—without dependencies, style collisions, or setup headaches.

Two lines to install

Drop it in. Let the sky do the rest.

Copy the module, add the custom element, and style your page with the resulting data-theme attribute.

<script type="module"
  src="/skyshift.js"></script>

<skyshift-toggle
  default-theme="system"
></skyshift-toggle>

Portable by default

Shadow DOM keeps the illustration protected from whatever CSS your website already uses.

  • No runtime dependencies
  • Saved theme preference
  • Keyboard and screen-reader friendly
  • Customizable with CSS variables
