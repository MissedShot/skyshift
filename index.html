01 · Installation

One file. Two lines.

Copy the script into your project and place the custom element wherever the theme control belongs. Skyshift updates data-theme and remembers the visitor's choice.

HTML
<script src="/skyshift.js" defer></script>

<skyshift-toggle default-theme="system"></skyshift-toggle>
02 · Details

Designed for real interfaces.

The illustration is expressive; the implementation stays deliberately ordinary and dependable.

  • PortableWorks in plain HTML and modern frameworks through the standard Custom Elements API.
  • AccessibleUses a native checkbox with switch semantics, visible focus, dynamic labels, and reduced-motion support.
  • PersistentStores the chosen theme locally and synchronizes multiple Skyshift instances on the same page.
  • CustomizableExposes focused CSS variables for size, day, night, clouds, sun, moon, and focus styling.
